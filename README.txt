A widget to display your Everyday Hero profile with donate button.

== Description ==

Display your Everyday Hero profile beautifully on your website with this widget.

= Features =
* Beautiful design
* Very simple and easy to use
* Just copy your profile URL and the widget manages everything else
* Performance optimised to ensure fast loading times

= What is Everyday Hero? =
[everydayhero](https://www.everydayhero.com) is a service that aims to re-define the modern giving experience. While many existing services allow people to raise and donate money to causes, everydayhero enables fundraising and volunteering whilst also providing a visual representation of the time, effort, and social engagement that goes into supporting a cause. Supporters can select a charity, start fundraising or volunteering, and begin tracking their Giving Footprint.

== Installation ==

= Installation =
1. Upload `everyday-hero-widget` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance > Widgets and drag \'Everyday Hero\' to your sidebar
4. Fill in the settings: your Everyday Hero profile url, and the option to show the Everyday Hero logo 

= Requirements =
* PHP 5.2.0 or later
* Wordpress 4.0 or later
